TETRIS





GamePlay

Space Bar: Start & Pause
Down Arrow: Move Down
Left Arrow: Move Left
Right Arrow: Move Right
Up Arrow: Rotate Shape

Tetris was built by Jhun de Andres & Aaron Careaga for the GameTime project in Module 4 at the Turing School. The game was implemented entirely in Javascript and jQuery using Lodash and HTML 5 canvas.
